Random Caching Based Cooperative Transmission in Heterogeneous Wireless Networks

发表于2021-07-12|更新于2021-07-16|论文记录
|字数总计:1.7k|阅读时长:6分钟|阅读量:

文章链接：https://ieeexplore.ieee.org/document/8295271

新的知识点

  1. small-scale fading（小尺度衰落）：包括由移动台和基站的相对运动造成多普勒频移，引起的时间选择性衰落和由多径引起的频率选择性衰落。（长距离或者长时间范围）

  2. large-scale path-loss（大尺度路径损失）：即由于移动台和基站空间上的距离以及无线电波传播过程中受到建筑物阻挡

    大尺度衰落又分为路径损耗和阴影衰落

摘要

本文，综合考虑了SBS缓存以及在下行链路大规模HetNet中的协作问题，以缓存分布为设计参数，在随机缓存条件下提出了两个SBS协作缓存传输策略，首先推导出了每种方案下成功传输概率的表达式，然后，在每个方案下，通过优化缓存分布来考虑成功传输概率最大化。

正文

介绍

  1. 联合协作传输的可用条件：
    • 严格同步
    • 好的CSI(channel state information)在所有协作基站都是满足的

​ 否则非联合传输更为可取，尤其是在轻负载的情况下。

  1. 应该考虑的点：

    • the cache size of each SBSs
    • the distances between the cooperative SBSs and the typical user
    • 不应只考虑一个MBS，这样其他MBSs的干扰就无法考虑进去
    • 其他文章获得的仅为局部最优解

  2. 本文所做的贡献：

    • 以缓存分布为设计参数，提出了两种随机缓存下的SBS协作传输方案，第一种方案采用了非相干的联合传输，第二种方案有效结合了非相干联合传输与BS silencing
    • 分析了成功传输概率
    • 通过结果可知，影响最佳缓存分布的是文件流行度，缓存大小以及SBSs的数量

系统模型

​ 考虑两层下行链路异构网络，其中一层是MBSs，被另一层为密度更高的SBSs所覆盖，两者在空间分布是两个独立均匀的泊松点过程，假设用户的分布也是一个独立的泊松点过程

​ 路径损失的参数设为>2，对于小尺度衰落，假设为瑞利衰落信道

文件n的流行度ana_nan​满足∑n=1Nan=1\sum_{n=1}^{N}a_n=1∑n=1N​an​=1，不失一般性，假设a1>a2>...>ana_1>a_2>...>a_na1​>a2​>...>an​

假设

  1. 用户的分布是独立PPP
  2. 所有文件大小都是一样的
  3. 文件流行度分布是相同的并且是先验已知的
  4. 文件流行度以较慢的时间尺度发展，并且可以采用各种学习方法来估计文件流行度随时间的变化
  5. 每个MBS没有配备缓存缓存，而是通过高容量的回程链路连接到核心网络
  6. SBSs之间以及SBSs与核心网络之间没有连接
  7. K个协作SBSs只服务一个用户u0u_0u0​

目标

探索在没有回程的情况下，SBSs缓存如何在没有SBSs回程的条件下提高性能
